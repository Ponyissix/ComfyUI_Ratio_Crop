# Git 初始化与代码提交计划

收到，我们先把当前的项目状态保存到 Git 仓库中。这是一个非常好的习惯，特别是在我们即将进行复杂的模型适配修复之前。

## 执行步骤

1.  **配置 .gitignore**：
    *   这是一个 ComfyUI 插件，我们**不应该**把下载的大型模型文件（如 `models/` 下的 `.pth` 和 `transformers` 权重）提交到 Git。
    *   同时排除常见的 Python 临时文件（`__pycache__`, `.venv`, `.DS_Store` 等）。

2.  **提交代码**：
    *   将所有代码文件（`js/`, `nodes.py`, `__init__.py`, `model_utils.py` 等）添加到 Git 暂存区。
    *   执行第一次提交 (Commit)，此时我们的工作区就是干净的了。

3.  **后续**：
    *   提交完成后，我们再回过头来专心解决那个 SAM-HQ 的加载报错问题。
